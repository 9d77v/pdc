# pdc
Personal Data Center.

# Quickstart
```
git clone git@github.com:9d77v/pdc.git
cd pdc

#amd64 machine
docker-compose up -d

#arm64 machine
docker-compose up -d -f docker-compose.arm64.yml
```
homepage http://localhost:8080

docs http://localhost:8080/docs
